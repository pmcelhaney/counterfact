Understand Counterfact in 30 seconds.

Counterfact converts OpenAPI into a running API simulator. It is stateful, type-safe, hot-reload friendly, and controllable with JavaScript in a built-in REPL.

  • Need realistic data flows? Shared context across all routes.
  • Need speed? Start with one command and edit handlers live.
  • Need confidence? Use patterns for testing, failure simulation, and proxying.
